# 锻刀收益计算器 (Blade Forging Profit Calculator)

一个用于计算锻刀期望收益的命令行工具。本工具旨在帮助用户基于实时价格数据，批量计算锻刀操作的期望收益，并生成详细的收益报告，为决策提供数据支持。

## 功能特性

-   **更新价格数据**：通过调用csqaq的api获取最新的材料与成品刀具市场价格。
-   **批量计算期望**：批量计算大部分箱子五合一锻刀方案的期望收益，便于对比分析。
-   **计算保本率**：给定成本和锻刀次数，通过枚举计算保本率，锻刀更安心。
-   **锻刀模拟**：输入你想要的产物类别和你的收货成本，模拟产出和收益。

## 快速开始

### 运行程序

1.  确保您的系统已安装 **Python**（建议使用 Python 3.8 或更高版本）。
2.  下载或克隆本项目代码到本地。
3.  在终端或命令行中，导航至程序所在目录。
4.  在config文件中添加你的csqaq网站的api_key并将ip添加进白名单，具体操作请查看[csqaq官方文档](https://docs.csqaq.com/)：
5.  运行主程序：
    ```bash
    python 锻刀模拟器.py
    ```

## 功能详细介绍

### 1. 更新价格数据
- 从 csqaq API 自动获取最新的市场价格信息
- 更新材料和产物的实时售价数据
- 确保所有计算基于最新的市场行情，提高分析准确性

### 2. 批量计算期望
- 读取大部分常见箱子的锻刀数据，批量计算全部期望和保本率等关键参数
- 配方采取默认五合一，自动添加磨损溢价系数
### 3. 计算保本率
- 分析特定箱子五合一锻刀配方的保本率
- 支持自定义成本和锻刀次数，计算对应保本率

### 4. 锻刀模拟器
- 模拟真实的锻刀过程，过手瘾的同时也对锻刀收益有切身体验
- 支持自定义材料成本和产物类别，进行五合一锻刀
- 提供产出统计和收益分析

### 5. 输出产物名称列表
- 生成完整的锻刀产物名称列表
- 方便用户快速查询程序中存储的产物名称
- 可以在源代码的name2case变量中修改成自己的惯用名

---

**注意**：本工具的计算结果仅为理论期望值，实际锻刀收益可能因市场波动、配方等众多因素而有所差异，仅供参考。
